# YEZI 网站功能说明

## 当前已实现

- 首页商品墙：按截图风格展示白底商品网格，支持顶部分类筛选入口。
- 商品详情：展示大图轮播、商品编号、价格、材质、尺码表、护理说明、模特信息和尺码选择。
- 购物车/结账：展示购物车商品、尺寸、数量、金额汇总，并支持加减数量。
- 订单创建：提交联系信息和收货地址后写入 SQLite 订单表，并清空当前购物车。
- 会话购物车：使用 `yezi_session` Cookie 区分访客购物车。
- 用户账户：保留注册、登录和 HttpOnly 会话 Cookie，当前前台下单不依赖登录。
- 地址簿：已有地址 API 与表结构，后续适合并入结账地址自动填充，而不是单独账户页。
- SKU 变体：商品支持尺码变体，购物车按尺码加入。
- 库存管理：变体库存、预留库存、库存流水。
- 库存锁定：下单时先锁定库存，支付成功后扣减库存。
- 订单状态机：`inventory_locked`、`paid`、`cancelled` 等状态。
- 商品后台：管理员可以创建商品、维护详情字段、查看库存和订单。
- 图片上传：后台支持上传图片到 `public/uploads/`。
- 支付入口：当前前台只保留 PayPal 按钮入口；PayPal 配置完整时可创建真实 PayPal 订单。
- SQLite 数据：应用启动时自动创建数据库表，并写入第一批种子商品。

## 页面

- `/`：商品列表页，可使用 `?category=new`、`?category=mens` 等参数过滤。
- `/products/[slug]`：商品详情页。
- `/cart`：购物车和结账页。
- `/login`：登录/注册页。
- `/admin`：后台管理页。默认管理员：`admin@yezi.local` / `admin123`。

## API

- `GET /api/products`：返回商品列表，支持 `category` 查询参数。
- `GET /api/cart`：返回当前会话购物车和金额汇总。
- `POST /api/cart`：加入购物车。
- `PATCH /api/cart`：修改购物车行数量，数量小于等于 0 时删除该行。
- `POST /api/checkout`：创建订单、锁定库存、创建支付记录。
- `POST /api/auth/register`：注册。
- `POST /api/auth/login`：登录。
- `POST /api/auth/logout`：退出。
- `GET /api/account/addresses`：读取地址簿。
- `POST /api/account/addresses`：新增地址。
- `POST /api/admin/products`：后台创建商品。
- `PATCH /api/admin/products`：后台更新商品。
- `PATCH /api/admin/inventory`：调整变体库存。
- `POST /api/admin/upload`：上传商品图。
